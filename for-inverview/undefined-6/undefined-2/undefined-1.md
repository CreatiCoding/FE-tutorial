# 데이터베이스의 이해

### 1. 파일 처리 시스템

파일 단위로 처리하여 각각의 응용프로그램이 직접 파일을 검색, 입력, 삭제, 수정을 수행

**특징**  
- 논리적 파일 구조를 물리적 파일 구조로 구현 필요  
- 따라서 응용 프로그래머에게 물리적 파일 구조에 대한 이해 필요  
**단점**  
- 데이터 독립성, 일관성, 무결성 문제 발생  
- 공유성, 사용 편의성 저하

### 2. 데이터베이스

중복된 데이터를 한 곳에 집약시켜 중복을 배제한 상태에서 데이터를 관리  
특징: 통합, 저장, 운, 공유
